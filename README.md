# ⚽️ 유니티를 이용한 3D 1vs1 축구게임 프로젝트 ⚽️


<div id="2"></div>

##  개요
- 프로젝트 이름 : 1vs1 Soccer Game
- 프로젝트 기간 : 2023.05 ~2023.06
- 팀원 : 박경민 , 최진현
- 프레임워크 : Unity


## 📲 주요 기능

- Player 1
   - 이동 : W A S D  
   - 슈팅 : Left Space bar
  
- Player 2
   - 이동 : 방향키
   - 슈팅 : Right Shift bar
  
- Unity 애니메이션 활용
